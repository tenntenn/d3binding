d3binding
=========

Data Binding plugin of simple-binding.js for d3.js.

Dependencies
-------------

+ [simple-binding](https://github.com/tenntenn/simple-binding.js)
+ [d3js](http://d3js.org/)

Build
-------------

    $ grunt

Usage
-------------

Look at [sample](http://jsfiddle.net/uedatakuya/89UsU/) on jsfiddle.
